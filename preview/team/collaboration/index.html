Collaboration

The present version of TARDIS would not have been possible if not for the work of the following active and past members of our community. To know more about members with specific roles, visit the Community Roles page.

GSoC/SoCIS Students

The Google Summer of Code (GSoC) is a global program organized by Google where students work with an open source organization on a programming project during Summers. Summer of Code In Space (SoCIS) is another similar program by ESA that aims at promoting open-source “space-related” software to students by letting them code new functionalities.
